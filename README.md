# 🌵 Drought Prediction using GRU Neural Network

Predict drought severity from 7 days of weather data using PySpark + PyTorch.

---

## What This Project Does

- Takes **7 days of weather data** as input
- Predicts a **drought severity score (0–5)** for the next day
- Uses **PySpark** to process 19.3M rows of historical US weather data
- Uses a **GRU neural network** (PyTorch) to learn time-series patterns

| Score | Meaning |
|---|---|
| 0 | No drought |
| 1 | Abnormally dry |
| 2 | Moderate drought |
| 3 | Severe drought |
| 4 | Extreme drought |
| 5 | Exceptional drought |

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/your-username/drought-prediction.git
cd drought-prediction
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Hadoop (Windows only — required for PySpark)

Download `winutils.exe` from [here](https://github.com/steveloughran/winutils) and place it in `C:\hadoop\bin\`

---

## How to Run

### Step 1 — Preprocess data
Run all cells in `01_data_preprocessing.ipynb`

Outputs: `deep_learning_data.parquet` + `scaler.pkl`

### Step 2 — EDA & feature selection
Run `02_eda_exploration.ipynb` to see correlation heatmaps

### Step 3 — Train the model
Run all cells in `03_model_training.ipynb`

Output: `drought_gru_model.pth`

### Step 4 — (Bonus) Run the API

```bash
cd api
uvicorn main:app --reload
```

Then open `http://localhost:8000/docs` to test predictions.
